# Java Array Programs Collection

This repository contains a collection of Java programs demonstrating various array operations and problem-solving techniques. These programs are designed for beginners to understand one-dimensional and two-dimensional arrays in Java.

## Programs Included

### 1. Array_creations.java

* Demonstrates the creation and usage of arrays.
* Accepts user input for a `String` array.
* Prints the elements using an enhanced `for-each` loop.
* Includes examples of integer, character, and string arrays.

### 2. problem01.java

* Calculates the sum and average of elements in an integer array.
* Uses the `length` property to determine the size of the array.

### 3. problem02.java

* Finds the maximum and minimum elements in an array.
* Computes the product of the maximum and minimum values.

### 4. problem03.java

* Separates array elements into even and odd numbers.
* Calculates and displays:

  * Sum of even numbers
  * Sum of odd numbers

### 5. problem04.java

* Counts the number of digits in each array element.
* Adds numbers with:

  * Even digit counts to `evensum`
  * Odd digit counts to `oddsum`

### 6. problem05.java

* Calculates the sum of digits for each array element.
* Determines whether the digit sum is even or odd.
* Adds the original number to the corresponding total.

### 7. twod_array.java

* Demonstrates the use of two-dimensional arrays.
* Copies elements from one matrix to another.
* Prints the resulting matrix.

## Concepts Covered

* One-Dimensional Arrays
* Two-Dimensional Arrays
* Enhanced For Loops
* User Input with `Scanner`
* Finding Maximum and Minimum Values
* Sum and Average Calculations
* Even and Odd Number Operations
* Digit Counting
* Sum of Digits
* Matrix Manipulation

## Technologies Used

* Java
* JDK 8 or later
* IntelliJ IDEA / Eclipse / VS Code

## How to Run

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:

   ```bash
   cd Java-Array-Programs
   ```

3. Compile the Java files:

   ```bash
   javac *.java
   ```

4. Run a program:

   ```bash
   java problem01
   ```

## Learning Outcomes

By working through these programs, you will learn:

* How to create and initialize arrays.
* How to iterate through arrays efficiently.
* How to perform common array operations.
* How to work with matrices in Java.
* How to solve beginner-level programming problems using arrays.
