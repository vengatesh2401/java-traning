## 📖 Problem Descriptions

### 1. Palindrome Number
- **Difficulty:** Easy
- **Concepts:** Math, Number Manipulation
- **Approach:** Reverse the number and compare it with the original number.

---

### 2. Power of Two
- **Difficulty:** Easy
- **Concepts:** Loops, Mathematics
- **Approach:** Continuously divide the number by 2 while it is divisible. If the final value is 1, it is a power of two.

---

### 3. Add Digits
- **Difficulty:** Easy
- **Concepts:** Loops, Digit Manipulation
- **Approach:** Repeatedly sum the digits until the result becomes a single digit.

---

### 4. Power of Three
- **Difficulty:** Easy
- **Concepts:** Mathematics
- **Approach:** Divide the number by 3 until it is no longer divisible. Check whether the final value is 1.

---

### 5. Power of Four
- **Difficulty:** Easy
- **Concepts:** Mathematics
- **Approach:** Divide the number by 4 until it is no longer divisible. Check whether the final value is 1.

---

### 6. Number of Steps to Reduce a Number to Zero
- **Difficulty:** Easy
- **Concepts:** Simulation
- **Approach:**
- If the number is even, divide it by 2.
- Otherwise, subtract 1.
- Count the operations until the number becomes 0.
