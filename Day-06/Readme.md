# Java Pattern Programs

A collection of beginner-friendly Java pattern programs designed to improve problem-solving skills, nested loops, and logical thinking.

## 📚 Programs Included

1. **Pattern Creation**

   * Prints a solid square pattern using `*`.

2. **Hollow Square**

   * Prints a hollow square pattern.

3. **Right Side Align Triangle**

   * Prints a right-aligned triangle using `*`.

4. **Left Side Align Triangle**

   * Prints an inverted left-aligned triangle.

5. **Pyramid Triangle**

   * Prints a pyramid pattern.

6. **Inverted Triangle**

   * Prints an inverted pyramid pattern.

7. **Diamond Triangle**

   * Prints a diamond-shaped pattern.

8. **Number Triangle**

   * Prints a triangle of increasing numbers.

9. **Floyd's Triangle**

   * Prints Floyd's Triangle using consecutive numbers.

10. **Binary Triangle**

    * Prints a binary pattern (`1` and `0`) based on row and column positions.

11. **Alphabet Solid Square Triangle**

    * Prints alphabets in a square pattern starting from `A`.

12. **Multiplication Table Pattern**

    * Prints a multiplication table in matrix form.

13. **Multiplication Pattern**

    * Prints a custom multiplication-based number pattern.

14. **Zigzag Triangle**

    * Prints a zigzag-style number pattern.

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/java-pattern-programs.git
```

2. Navigate to the project directory:

```bash
cd java-pattern-programs
```

3. Compile a Java file:

```bash
javac pattern_creation.java
```

4. Run the program:

```bash
java pattern_creation
```
