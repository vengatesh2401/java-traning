# Java Searching and Sorting Programs

This repository contains a collection of Java programs that demonstrate fundamental searching and sorting algorithms, along with operations on one-dimensional and two-dimensional arrays. These programs are ideal for beginners who want to strengthen their understanding of Data Structures and Algorithms (DSA) using Java.

## Programs Included

### 1. bubble_sort.java

* Implements Bubble Sort to arrange elements in ascending order.
* Demonstrates swapping elements using nested loops.
* Prints the sorted array.

### 2. bubble_sorting_using_function.java

* Implements Bubble Sort using a separate method.
* Highlights the concept of modular programming.
* Shows how arrays are passed to functions in Java.

### 3. char_searching.java

* Performs a linear search for a character within a string.
* Converts a string into a character array using `toCharArray()`.
* Returns whether the target character is found.

### 4. linear_search.java

* Implements Linear Search on an integer array.
* Accepts user input for:

  * Array size
  * Target element
  * Array elements
* Returns the index of the target element or `-1` if not found.

### 5. min_value_max_value.java

* Finds the minimum and maximum values in a two-dimensional array.
* Demonstrates traversal of matrices using nested loops.

### 6. number_2_dimension_array.java

* Searches for a target element in a two-dimensional array.
* Returns a boolean value indicating whether the element exists in the matrix.

### 7. selection_sorting.java

* Implements the Selection Sort algorithm.
* Finds the minimum element in the unsorted portion of the array.
* Swaps elements to produce a sorted array.

## How to Run

1. Compile the Java files:

   ```bash
   javac *.java
   ```

2. Execute a program:

   ```bash
   java linear_search
   ```
