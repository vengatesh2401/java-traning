# Java Programming Practice - Day 03

This repository contains Java practice programs covering fundamental programming concepts such as loops, conditional statements, switch cases, operators, and number-based problems. These programs are designed for beginners to strengthen their Java programming skills.

---

## 📁 Project Structure

```
day03/
│── calculator.java
│── loop.java
│── main.java
│── nested_if.java
│── number_cringing.java
│── print_odd_number.java
│── switch_case.java
│── ternary_operator.java
│── vowel_consonant.java
└── while_loop.java
```

---

## 📚 Programs Included

### 1. calculator.java
A simple calculator using the **Switch Case** statement.

**Operations Supported**
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Modulus (%)

**Concepts Used**
- Scanner Class
- Switch Statement
- Arithmetic Operators

---

### 2. loop.java
Demonstrates the use of a **for loop**.

**Features**
- Prints numbers from 0 to 9.
- Shows an alternative way of writing a for loop.

**Concepts Used**
- For Loop
- Loop Syntax

---

### 3. main.java
Basic Java program that accepts two integers from the user.

**Concepts Used**
- Scanner Class
- User Input

---

### 4. nested_if.java
Checks divisibility of a number using **Nested If Statements**.

**Concepts Used**
- Nested If
- Divisibility Check

---

### 5. number_cringing.java
Performs number-based operations.

**Features**
- Reverse a Number
- Count Number of Digits

**Concepts Used**
- While Loop
- Arithmetic Operations

---

### 6. print_odd_number.java
Contains multiple loop practice examples.

Examples include:
- Print Odd Numbers
- Sum of Numbers
- Factorial Calculation
- ASCII Values
- Multiplication Table

**Concepts Used**
- For Loop
- Mathematical Logic

---

### 7. switch_case.java
Prints the day of the week using a Switch Case.

| Input | Output |
|--------|--------|
|1|Sunday|
|2|Monday|
|3|Tuesday|
|4|Wednesday|
|5|Thursday|
|6|Friday|
|7|Saturday|

---

### 8. ternary_operator.java
Demonstrates the **Ternary Operator**.

Example:
- Check Even or Odd
- Adult or Minor (Commented)

**Concepts Used**
- Conditional Operator (`? :`)

---

### 9. vowel_consonant.java
Determines whether a character is a vowel or consonant.

**Features**
- Uses ASCII values
- Uses Switch Case

**Concepts Used**
- Character Type Casting
- Switch Statement

---

### 10. while_loop.java
Prints numbers in reverse order using a While Loop.

Example:

Input:
```
5
```

Output:
```
5 4 3 2 1
```
