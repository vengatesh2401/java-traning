# Java String Methods Programs

This repository contains basic Java programs that demonstrate various String and StringBuilder methods. These examples help beginners understand common string operations in Java.

## Programs Included

### 1. Strings.java
Demonstrates:
- String comparison using `==`
- String comparison using `equals()`
- StringBuilder comparison
- Converting StringBuilder to String

### 2. Len.java
Demonstrates:
- Finding the length of a string using `length()`

### 3. Equal.java
Demonstrates:
- Comparing strings using `equals()`
- Comparing strings ignoring case using `equalsIgnoreCase()`

### 4. Compare.java
Demonstrates:
- Lexicographical comparison using `compareTo()`

### 5. Contains.java
Demonstrates:
- Checking whether a string contains a substring using `contains()`

### 6. Char_Methods.java
Demonstrates:
- Accessing characters using `charAt()`
- Converting a string into a character array using `toCharArray()`

### 7. Replace.java
Demonstrates:
- Replacing characters using `replace()`
- Replacing text using `replaceAll()`
- Replacing first occurrence using `replaceFirst()`

### 8. Split.java
Demonstrates:
- Splitting strings using `split()`
- Iterating through resulting arrays

### 9. String_Builder_Append.java
Demonstrates:
- Appending text using `StringBuilder.append()`

### 10. problem_palidrome.java
Demonstrates:
- Reversing a string using a character array

### 11. Palidrome_check.java
Demonstrates:
- Checking whether a string is a palindrome

### 12. Insert.java
Reserved for:
- StringBuilder `insert()` method implementation

## Prerequisites

- Java JDK 8 or higher
- Any Java IDE (IntelliJ IDEA, Eclipse, VS Code) or Command Prompt

## How to Run

1. Compile the program:

```bash
javac FileName.java
```

2. Run the program:

```bash
java FileName
```

Example:

```bash
javac Len.java
java Len
```
