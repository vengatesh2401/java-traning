# Java Operators Practice

This repository contains Java programs demonstrating different types of operators and basic programming concepts.

## Files Included

| File Name | Description |
|-----------|-------------|
| `main.java` | Basic arithmetic operators and increment/decrement examples. |
| `assignment.java` | Assignment operators (`=`, `+=`, `-=`, `*=`, `/=`, `%=` etc.). |
| `relational.java` | Relational operators (`==`, `!=`, `>`, `<`, `>=`, `<=`). |
| `logical.java` | Logical operators (`&&`, `||`, `!`). |
| `bitwise.java` | Bitwise operators (`&`, `|`, `^`, `<<`, `>>`). |
| `condition.java` | Basic `if-else` conditional statements. |
| `practice.java` | Practice programs including salary calculation, temperature conversion, loan eligibility, average marks, and more. |

## Topics Covered

- Java Basics
- Scanner Class
- User Input
- Arithmetic Operators
- Assignment Operators
- Relational Operators
- Logical Operators
- Bitwise Operators
- Conditional Statements
- Practice Programs

## Requirements

- Java JDK 8 or above
- Any Java IDE (VS Code, IntelliJ IDEA, Eclipse) or Command Prompt

## Compile

```bash
javac *.java
```

## Run

```bash
java main
```

Example:

```bash
java assignment
java relational
java logical
java bitwise
java condition
java practice
```

## Learning Objectives

- Understand Java operators.
- Learn how to take user input using `Scanner`.
- Practice conditional statements.
- Improve problem-solving using simple Java programs.
