# Java Basics

This repository contains simple Java programs demonstrating user input and type casting concepts.

## Files

### 1. `main.java`
This program demonstrates:
- Taking user input using `Scanner`
- Reading different Java data types:
  - int
  - byte
  - short
  - long
  - char
  - float
  - double
  - String
- Printing values to the console

> **Note:** This file contains some syntax errors that need to be fixed before it can compile successfully.

### 2. `type_casting.java`
This program demonstrates:
- Explicit type casting
- Conversion between different primitive data types
- Data loss caused by narrowing conversions
- Printing the converted values

## Requirements

- Java JDK 8 or later
- Any Java IDE (VS Code, IntelliJ IDEA, Eclipse) or Command Prompt

## How to Compile

```bash
javac main.java
javac type_casting.java
```

## How to Run

```bash
java Main
```

```bash
java type_casting
```

## Topics Covered

- Java Basics
- Scanner Class
- User Input
- Primitive Data Types
- Type Casting
- Console Output
